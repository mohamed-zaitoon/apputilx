ðŸ“˜ AppUtils Documentation

A complete guide on how to use AppUtils in Android projects (Java / Kotlin).

ðŸ”¹ Download & Setup

Add JitPack repository and dependency in your build.gradle:

repositories {
    maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.mohamed-zaitoon:apputils:1.2.1'
}

Ensure Java 21 compatibility:

android {
    compileOptions {
        sourceCompatibility JavaVersion.VERSION_21
        targetCompatibility JavaVersion.VERSION_21
    }
}

For Kotlin DSL:

plugins {
    id("com.android.application") version "8.12.0" apply false
    id("org.jetbrains.kotlin.android") version "2.2.10" apply false
}

android {
    compileOptions {
        sourceCompatibility = JavaVersion.VERSION_21
        targetCompatibility = JavaVersion.VERSION_21
    }
}

repositories {
    maven { url = uri("https://jitpack.io") }
}

dependencies {
    implementation("com.github.mohamed-zaitoon:apputils:1.2.1")
}

ðŸ”¹ Initialization

Call initialize() once inside onCreate().

Java 
AppUtils.initialize(this);
Kotlin 
AppUtils.initialize(this)

ðŸ”¹ Toast

Java 
AppUtils.showToast("Hello World");
AppUtils.showToast("Long Toast", true);
Kotlin 
AppUtils.showToast("Hello World")
AppUtils.showToast("Long Toast", long = true)

ðŸ”¹ Snackbar

Java 
AppUtils.showSnackbar(view, "Message", null, "OK", v -> {
    AppUtils.showToast("Clicked!");
});
Kotlin 
AppUtils.showSnackbar(rootView, "Message", null, "OK") {
    AppUtils.showToast("Clicked!")
}

ðŸ”¹ Connectivity

Java 
boolean connected = AppUtils.isConnected();
AppUtils.addConnectionListener(isOnline -> {
    AppUtils.showToast("Network changed: " + isOnline);
});
Kotlin 
val connected = AppUtils.isConnected
AppUtils.addConnectionListener { isOnline ->
    AppUtils.showToast("Network changed: $isOnline")
}

ðŸ”¹ Set Locale / Language

Change the app language at runtime or on startup.

Java 
// Set language at runtime
AppUtils.setLanguage("ar"); // e.g., "ar" for Arabic, "en" for English
// Optionally recreate activity to apply changes
recreate();
Kotlin 
// Set language at runtime
AppUtils.setLanguage("ar") // e.g., "ar" for Arabic, "en" for English
// Optionally recreate activity to apply changes
recreate()

To apply the saved language automatically when the activity starts:

Kotlin / Java 
@Override
protected void attachBaseContext(Context newBase) {
    super.attachBaseContext(AppUtils.attachBaseContext(newBase));
}
